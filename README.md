# Portal ze zwierzętami do adopcji "Adoptable"

## Opis projektu:

Aplikacja “Adoptable” to portal, którego główną funkcją będzie możliwość publikacji profili zwierząt do adopcji przez schroniska, a następnie aplikowanie o adopcję przez potencjalnych adoptujących.

## Możliwości aplikacji:

### Schronisko

- Profil schroniska umożliwiający dodanie informacji o lokalizacji schroniska i możliwych sposobach kontaktu
- Publikowanie profili zwierząt do adopcji
- Zaakceptowanie bądź odrzucenie aplikacji o adopcję

### Potencjalny adoptujący

- Wyszukiwanie zwierząt do adopcji z możliwością filtracji wyników
- Aplikowanie o adopcję wybranego zwierzęcia
- Możliwość wypełnienia ankiety, której wyniki pozwolą na zasugerowanie odpowiedniego zwierzęcia do adopcji

## Wykorzystane technologie:

- Backend: Django
- Frontend: React JS
- Baza danych: PostgreSQL
